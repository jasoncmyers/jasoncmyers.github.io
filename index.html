Portfolio: Some Recent Projects

Nand to Tetris

A 12-project course which begins with only NAND logic gates and uses them to build increasingly complex logic and memory circuits leading up to a simple van Neumann-architecture computer.

The second half of the course began by implementing an assembly language and aseembler for that architecture. From there, it went to successively higher abstractions, including a stack-based virtual machine emulator, a compiler for a basic high-level language, and a series of OS libraries for the virtual machine.

Tools used: C#, assembly, Jack (language implemented during the project)

.ser File Reader

ServoStik MaLa plugin

A DLL plugin file for the MaLa front end to MAME (Multiple Arcade Machine Emulator) that controls Ultimarc ServoStik joysticks.

Written as part of the construction of a cocktail-table-style arcade emulator, this plugin handles event calls from a launcher and reads from a list of games with 4-way or 8-way joystick configurations. When a game is launched, it signals a servo controller in the joystick to rotate a restrictor plate and limit motion appropriately, giving an authentic control scheme without any effort from the player.

Tools used: C++, Ultimarc SDK

TEM HTML/JS Applets

A series of applets written to automate tasks and/or add functionality to the FEI (now Thermo Scientific) Tecnai line of transmission electron microscopes.

These scripts are designed to circumvent the limitations of the microscope computers (Win2k, almost no usable UI frameworks) by accessing the microscope hardware as an ActiveX object, interfacing via JavaScript code, and providing simple HTML-driven interfaces. Each provides otherwise inaccessible functionality or automation for the Tecnai TEM.

Tools used: HTML, JavaScript, ActiveX, Gatan DigitalMicrograph scripting
