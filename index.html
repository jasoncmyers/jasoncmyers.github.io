About Me

Portfolio: Some Recent Projects

Nand to Tetris

A 12-project course which begins with only NAND logic gates and uses them to build increasingly complex logic and memory circuits leading up to a simple van Neumann-architecture computer.

The second half of the course had us construct an assembler and from there went to successively higher abstractions, including a virtual machine emulator, compiler for a simple language, and a series of OS libraries for the virtual machine.

Tools used: C#, assembly, Jack (language implemented during the project)

Quote Generator

A FreeCodeCamp project website that selects a random quote from a list and displays it. Also includes a link that allows a user to tweet the quote.

Tools used: HTML5, CSS3, Bootstrap3, jquery, javascript

.ser File Reader

A library implemented to read the .ser files produced by the ESVision/Tia transmission electron microscopy software.

The library allows the reading, manipulation, and writing of both individual images and image stacks.

Tools used: C++

Data Analysis: Severe Weather Events

A project for the Coursera Reproducible Research course. It involves an analysis of various measures of economic damages from severe in the United States.

Tools used: R, R Markdown
